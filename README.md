# 🎧 FluxStudio

FluxStudio is a browser-based **Digital Audio Workstation (DAW)** combined with a **music recognition system**, designed to allow users to create, edit, and analyze audio directly in the web. It provides a modern, scalable interface similar to professional tools like BandLab, enabling multi-track music production without requiring any software installation.

The platform allows users to upload or record audio, arrange clips on a timeline, apply effects such as EQ, reverb, and delay, and control playback with features like BPM adjustment, snapping, and zoom. Additionally, it includes a Shazam-like feature that detects music using microphone input and displays basic song information.

FluxStudio also includes a project management system where users can save, load, and manage their audio projects. All project data (tracks and clips) is stored in a database, while audio files are securely stored in cloud storage, ensuring persistence across sessions.

## 🛠️ Technologies Used

- **Frontend:** React, TypeScript, Vite  
- **Audio Engine:** Web Audio API  
- **Backend & Database:** Supabase (PostgreSQL)  
- **Storage:** Supabase Storage  
- **Authentication:** Supabase Auth  
- **Routing & UI:** React Router, modern CSS  

FluxStudio demonstrates a full-stack approach to building a real-time, interactive audio production application in the browser.
